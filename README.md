# OC Cal Alumni Club — Event Planner

A single self-contained HTML page that walks OC Cal Alumni Club board members through planning an event end to end: concept, logistics, RSVP platform (Eventbrite/Partiful), CAA grant funding, an auto-generated execution timeline with a downloadable calendar invite, and ready-to-post content (email, Instagram, Facebook, LinkedIn) with direct links to post.

No backend, no build step, no dependencies — it's one file (`index.html`).

## Deploy on Vercel

1. Push this repo to GitHub (this file plus `index.html` at the root).
2. Go to [vercel.com](https://vercel.com) → **Add New Project** → import this repo.
3. Framework preset: **Other**. No build command or output directory needed — Vercel serves `index.html` as-is.
4. Click **Deploy**. You'll get a live URL (e.g. `your-project.vercel.app`) in under a minute.
5. To update: edit `index.html`, commit, push — Vercel auto-redeploys.

## Local preview

Just open `index.html` directly in a browser. No server required.
